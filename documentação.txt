# 🎨 Sistema de Produção - Bordados

Sistema modernizado de gestão de produção com arquitetura cliente-servidor.

## 🚀 Características

### ✨ Design Moderno
- Interface responsiva e intuitiva
- Tema com gradientes e animações suaves
- Totalmente adaptável para mobile, tablet e desktop

### 👥 Controle de Acesso
- **Gerente**: Acesso completo (exportar, excluir registros)
- **Colaborador**: Pode adicionar e editar apenas suas produções

### 🔧 Funcionalidades
- ✅ Registro de produção de bordados
- ✏️ Edição de registros (correção de erros)
- 📊 Dashboard com estatísticas em tempo real
- 💾 Exportação para Excel com formatação
- 🔐 Sistema de login seguro
- 📱 Interface 100% responsiva

---

## 📋 Pré-requisitos

- Python 3.8 ou superior
- Navegador web moderno (Chrome, Firefox, Edge, Safari)

---

## 🛠️ Instalação

### 1. Instalar Dependências

```bash
# Instalar pacotes Python
pip install -r requirements.txt
```

### 2. Estrutura de Arquivos

Organize os arquivos da seguinte forma:

```
projeto/
│
├── app.py                    # Backend Flask
├── index.html                # Frontend
├── requirements.txt          # Dependências
│
├── dados_producao.json       # Criado automaticamente
├── bordadores.json           # Criado automaticamente
└── usuarios.json             # Criado automaticamente
```

---

## ▶️ Executando o Sistema

### 1. Iniciar o Backend

```bash
python app.py
```

O servidor iniciará em `http://localhost:5000`

### 2. Abrir o Frontend

- **Opção 1**: Abra o arquivo `index.html` diretamente no navegador
- **Opção 2**: Use um servidor HTTP simples:

```bash
# Python 3
python -m http.server 8000

# Acesse: http://localhost:8000
```

## 🎨 Personalização

### Cores do Tema

Edite as variáveis CSS em `index.html`:

```css
:root {
    --primary: #6366f1;      /* Cor principal */
    --success: #10b981;      /* Cor de sucesso */
    --danger: #ef4444;       /* Cor de perigo */
    --warning: #f59e0b;      /* Cor de aviso */
}
```

### Adicionar Novos Usuários

Edite o arquivo `usuarios.json` (criado após primeiro login):

```json
{
  "novo_usuario": {
    "senha": "hash_da_senha",
    "tipo": "colaborador",
    "nome": "Nome do Usuário"
  }
}
```

Para gerar hash de senha:

```python
from werkzeug.security import generate_password_hash
print(generate_password_hash("sua_senha"))
```

## 🐛 Troubleshooting

### Erro de CORS
- Certifique-se de que o backend está rodando
- Verifique se a URL da API está correta no frontend

### Erro ao Exportar Excel
- Verifique se a pasta `exports/` existe
- Confirme que tem permissões de escrita

### Dados não Aparecem
- Verifique o console do navegador (F12)
- Confirme que os arquivos JSON estão sendo criados
